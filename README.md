# Deal_Platform

校园二手交易系统，第一阶段已完成作业基础要求。

## 启动方式

1. 在 IDEA 中运行 `com.example.dealplatform.DealPlatformApplication`。
2. 访问：`http://localhost:8080`

## 第一阶段功能

- 用户注册、登录、退出
- Session 会话保持
- 未登录访问核心页面自动跳转登录页
- 管理员和普通用户角色区分
- 管理员后台权限控制
- 普通用户前台权限控制
- 商品新增、查询、修改、删除
- 商品分页展示
- 前端基础非空和格式校验
- 后端参数二次校验
- 个人信息查看与修改
- 密码修改
- MySQL 建库建表和初始数据脚本

## 第二阶段功能

- 商品分类管理
- 商品图片上传和预览
- 商品审核、上架、下架
- 商品搜索、分类筛选、排序
- 商品收藏和浏览记录
- 商品留言咨询与卖家回复
- 创建订单
- 我的购买订单、我的出售订单
- 买家取消订单
- 卖家确认交易完成
- 订单状态日志
- 系统公告
- 站内消息通知

## 第三阶段功能

- 订单评价和评价管理
- 商品举报和后台举报处理
- 操作日志记录与查询
- 后台数据统计卡片
- 商品分类统计图
- 订单状态统计图
- 商品批量删除
- 商品数据导出 Excel
- 首页轮播图展示与管理
- 文件资源展示效果优化
